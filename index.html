操作指南

生成流程

  • 在工作台输入提示词，选择尺寸、格式、数量、背景、审核和质量后提交。
  • 只输入提示词会使用文生图；添加参考图会自动使用图文编辑。
  • 提交后任务会进入图库，工作台保持干净用于继续创作。

图库管理

  • 在顶部切换工作台和图库，图库徽标会显示当前任务数量。
  • 图库支持按状态、收藏和关键词筛选生成记录。
  • 失败任务可重试，也可以通过顶部删除按钮批量清除。
  • 图库会自动保存到本地缓存，刷新页面后会恢复已生成图片。

设置与日志

  • 设置中填写 API 地址、密钥和模型 ID，模型默认使用 gpt-image-2。
  • 请求日志会记录接口、状态和错误信息，便于排查问题。
  • Ctrl + Enter 可快速提交当前提示词。
